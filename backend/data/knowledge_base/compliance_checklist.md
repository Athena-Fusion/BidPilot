# 投标合规检查清单

## 1. 资格材料检查

- [ ] 营业执照副本复印件
- [ ] 财务审计报告（近两年）
- [ ] 依法缴纳税收证明
- [ ] 依法缴纳社保证明
- [ ] 无重大违法记录声明
- [ ] 特定资格证明（CMMI、ISO等）
- [ ] 业绩证明材料（合同关键页、验收报告）

## 2. 实质性条款检查

- [ ] 逐条响应★号技术参数
- [ ] 技术方案覆盖全部功能需求
- [ ] 服务周期满足要求
- [ ] 质保期满足要求

## 3. 报价检查

- [ ] 报价不超过预算
- [ ] 报价不低于成本
- [ ] 分项报价合计与总报价一致
- [ ] 报价表签字盖章

## 4. 保证金检查

- [ ] 保证金金额正确
- [ ] 交纳方式符合要求
- [ ] 到账时间在截止时间前

## 5. 签章检查

- [ ] 投标文件加盖公章
- [ ] 法定代表人签字或盖章
- [ ] 报价表签字盖章
- [ ] 资格文件逐页盖章

## 6. 业绩证明检查

- [ ] 业绩数量满足要求
- [ ] 业绩金额满足要求
- [ ] 业绩时间范围满足要求
- [ ] 合同关键页完整（首页、金额页、签字页）

## 7. 技术参数检查

- [ ] ★号参数逐条响应
- [ ] 响应表覆盖全部技术要求
- [ ] 无未说明的负偏离
- [ ] 正偏离有证据支撑

## 8. 偏离表检查

- [ ] 技术偏离表完整
- [ ] 商务偏离表完整
- [ ] 负偏离有合理说明
- [ ] 无遗漏项

## 9. 文件格式检查

- [ ] 投标文件为PDF格式
- [ ] 电子文件已上传
- [ ] 纸质文件密封递交
- [ ] 文件命名符合要求

## 10. 其他检查

- [ ] 投标截止时间前递交
- [ ] 未以联合体形式投标（如不允许）
- [ ] 未分包（如不允许）
- [ ] 投标文件副本份数符合要求
