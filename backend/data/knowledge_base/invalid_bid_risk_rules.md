# 废标风险规则库

## 1. 价格废标风险

### 1.1 报价超预算
- 规则：投标报价超过采购预算的，按无效投标处理
- 检查点：报价金额 vs 预算金额
- 处理：报价必须严格控制在预算以内

### 1.2 报价明显低于成本
- 规则：报价明显低于其他投标人报价，且不能合理说明的
- 检查点：报价是否低于平均报价的70%
- 处理：准备成本说明材料

## 2. 资格废标风险

### 2.1 资格条件不满足
- 规则：投标人必须满足全部资格条件
- 检查点：逐条核对资格要求
- 处理：提前准备资格证明文件

### 2.2 业绩不足
- 规则：类似项目业绩数量或金额不满足要求
- 检查点：业绩合同数量、金额、时间范围
- 处理：整理全部相关业绩，确保合同关键页完整

## 3. 格式废标风险

### 3.1 未签字盖章
- 规则：投标文件须加盖公章和法定代表人签字
- 检查点：逐页检查盖章、签字
- 处理：制作盖章检查清单

### 3.2 保证金缺失
- 规则：未按规定交纳投标保证金
- 检查点：保证金金额、交纳方式、到账时间
- 处理：提前确认保证金到账

### 3.3 文件格式错误
- 规则：投标文件格式不符合要求
- 检查点：PDF格式、电子上传、密封要求
- 处理：按招标文件要求逐项检查

## 4. 实质性条款风险

### 4.1 未响应★号参数
- 规则：带★号条款为实质性要求，必须响应
- 检查点：所有★号条款是否逐条响应
- 处理：技术方案和响应表逐条覆盖★号条款

### 4.2 负偏离未说明
- 规则：不接受负偏离或负偏离未说明
- 检查点：偏离表中是否有未说明的负偏离
- 处理：避免负偏离，必要时提供合理说明

## 5. 商务条款风险

### 5.1 服务周期不满足
- 规则：建设周期须满足招标要求
- 检查点：承诺工期 vs 要求工期
- 处理：合理安排项目计划

### 5.2 付款方式不接受
- 规则：须接受招标文件规定的付款方式
- 检查点：付款比例和节点
- 处理：商务响应中明确接受

## 6. 其他风险

### 6.1 联合体投标
- 规则：不接受联合体投标时，联合体投标无效
- 处理：确认是否允许联合体

### 6.2 分包限制
- 规则：不接受分包时，分包方案无效
- 处理：确认是否允许分包
