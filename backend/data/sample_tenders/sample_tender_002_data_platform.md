# 某区数据中台建设项目招标文件

## 第一章 招标公告

### 1.1 项目基本信息

- 项目名称：某区数据中台建设项目
- 采购人：某区政务服务数据管理局
- 采购方式：公开招标
- 项目预算：人民币800万元（投标报价超过采购预算的，按无效投标处理）
- 投标截止时间：2026年8月1日 10:00
- 开标时间：2026年8月1日 10:00
- 建设周期：合同签订后120日内完成建设并上线试运行
- 交付地点：采购人指定机房
- 投标保证金：人民币8万元
- 采购范围：数据中台平台建设、数据治理、数据接入、系统集成、培训与运维服务

### 1.2 采购内容

1. 数据治理平台
2. 数据目录系统
3. 数据交换共享平台
4. 数据质量管理系统
5. 数据安全管控平台
6. 标准接口服务

## 第二章 投标人资格要求

### 2.1 基本资格条件

1. 具有独立承担民事责任能力的在中华人民共和国境内注册的法人
2. 具有良好的商业信誉和健全的财务会计制度
3. 具有履行合同所必需的设备和专业技术能力
4. 有依法缴纳税收和社会保障资金的良好记录
5. 参加政府采购活动前三年内，在经营活动中没有重大违法记录

### 2.2 特定资格条件

1. 投标人须具有CMMI3级及以上认证或同等软件能力认证
2. 近三年（2023年1月1日至今）具有至少3个合同金额不低于300万元的数据治理或数据中台相关项目业绩
3. 项目团队中至少有2人具有数据治理相关认证（如DAMA CDMP）
4. 投标人须具有ISO 27001信息安全管理体系认证
5. 须具有涉密信息系统集成资质或等保相关项目经验

### 2.3 联合体投标

本项目不接受联合体投标。

## 第四章 招标技术要求

### 4.1 数据治理平台

1. ★支持元数据管理，包括技术元数据和业务元数据
2. 支持数据标准管理，支持国标、行标和自定义标准
3. ★支持数据质量规则配置和自动检测
4. 支持数据资产目录和资产盘点
5. 支持数据血缘分析和影响分析
6. 支持数据生命周期管理

### 4.2 数据目录系统

1. 支持数据资源目录编目
2. 支持目录分类和标签管理
3. 支持数据资源检索和预览
4. 支持数据资源申请和审批流程
5. ★符合政务数据目录标准规范

### 4.3 数据交换共享平台

1. ★支持不少于10种数据源接入（关系型数据库、NoSQL、文件、API等）
2. 支持全量/增量/实时数据同步
3. 支持数据交换任务调度和监控
4. ★提供标准REST API接口，支持OAuth2.0认证
5. 支持数据交换日志和审计

### 4.4 数据质量管理系统

1. ★支持完整性、准确性、一致性、及时性、唯一性等质量维度检测
2. 支持自定义质量规则
3. 支持质量检测报告生成
4. 支持质量问题工单处理闭环
5. 支持质量趋势分析和预警

### 4.5 数据安全要求

1. ★敏感数据加密存储和传输
2. 支持数据脱敏（静态脱敏和动态脱敏）
3. 支持数据访问控制和权限管理
4. 支持数据操作审计日志
5. ★满足等保三级要求

### 4.6 接口标准

1. ★提供标准RESTful API接口文档（OpenAPI 3.0规范）
2. 接口支持OAuth2.0或API Key认证
3. 接口支持速率限制
4. 接口版本管理

## 第五章 商务条款

### 5.1 服务周期

合同签订后120日内完成建设并上线试运行，试运行期30天。

### 5.2 付款方式

1. 合同签订后支付30%
2. 系统上线试运行后支付40%
3. 验收合格后支付25%
4. 质保期满后支付5%

### 5.3 质保期

验收合格后2年免费质保。

### 5.4 培训要求

1. 提供不少于8天的现场培训
2. 提供培训教材和操作手册
3. 培训覆盖数据管理员、业务操作员、接口开发者

### 5.5 售后服务

1. 质保期内7×24小时技术支持
2. 故障响应时间不超过2小时
3. 重大故障4小时内到达现场

### 5.6 知识产权

项目成果知识产权归采购人所有。

### 5.7 保密条款

投标人须对项目相关数据和技术信息严格保密，保密期限为合同终止后5年。
