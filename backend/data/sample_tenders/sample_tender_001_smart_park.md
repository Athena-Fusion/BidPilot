## 第二章 投标人资格要求

### 2.1 基本资格条件

1. 具有独立承担民事责任能力的在中华人民共和国境内注册的法人
2. 具有良好的商业信誉和健全的财务会计制度
3. 具有履行合同所必需的设备和专业技术能力
4. 有依法缴纳税收和社会保障资金的良好记录
5. 参加政府采购活动前三年内，在经营活动中没有重大违法记录

### 2.2 特定资格条件

1. 投标人须具有计算机信息系统集成及服务资质或软件能力成熟度模型集成（CMMI）3级及以上认证

3. 项目经理须具有PMP认证或信息系统项目管理师证书，且具有3年以上项目管理经验
4. 投标人须具有ISO 27001信息安全管理体系认证

### 2.3 联合体投标

本项目不接受联合体投标。

## 第四章 招标技术要求

### 4.1 统一门户系统

1. ★系统须支持不少于500个并发用户访问
2. 支持单点登录（SSO）和统一认证
3. 支持多终端适配（PC、平板、手机）
4. 页面加载时间不超过3秒
5. 支持个性化工作台配置

### 4.2 设备接入管理平台

1. ★支持不少于200种设备协议接入（包括但不限于Modbus、BACnet、OPC UA、MQTT、HTTP）
2. 支持设备自动发现和注册
3. 支持设备状态实时监控和告警
4. 支持设备数据采集频率不低于1秒
5. 提供标准API接口供第三方系统调用

### 4.3 数据可视化大屏

1. 支持实时数据展示，刷新频率不低于5秒
2. 支持不少于10个可视化组件
3. 支持大屏自适应布局
4. 支持数据钻取和联动分析

### 4.4 权限管理系统

1. 支持基于角色的访问控制（RBAC）
2. 支持组织架构和岗位管理
3. 支持功能权限和数据权限分离
4. 支持权限变更实时生效

### 4.5 日志审计系统

1. ★记录所有用户操作日志，日志保留不少于180天
2. 支持日志检索和统计分析
3. 日志数据不可篡改
4. 支持异常行为告警

### 4.6 国产化适配要求

1. ★须适配国产CPU（飞腾/鲲鹏/龙芯至少一种）
2. ★须适配国产操作系统（银河麒麟/统信UOS至少一种）
3. ★须适配国产数据库（达梦/人大金仓/海量数据至少一种）
4. 须适配国产中间件（东方通/宝兰德至少一种）

### 4.7 数据安全要求

1. ★敏感数据须加密存储，加密算法须符合国家密码管理要求
2. ★数据脱敏须支持姓名、手机号、身份证号等常见脱敏规则
3. 数据分类分级管理
4. 数据导出须审批并记录日志
5. 须满足等保二级要求

## 第五章 商务条款

### 5.1 服务周期

合同签订后90日内完成建设并上线试运行，试运行期30天，试运行期满验收合格后进入运维期。

### 5.2 付款方式

1. 合同签订后支付合同总价的30%
2. 系统上线试运行后支付合同总价的40%
3. 验收合格后支付合同总价的25%
4. 质保期满后支付合同总价的5%

### 5.3 质保期

质保期为验收合格后1年，质保期内免费维护。

### 5.4 培训要求

1. 提供不少于5天的现场培训
2. 提供完整的培训教材和操作手册
3. 培训覆盖系统管理员、业务操作员

### 5.5 售后服务

1. 质保期内7×24小时技术支持
2. 故障响应时间不超过2小时
3. 重大故障4小时内到达现场

### 5.6 知识产权

1. 项目成果知识产权归采购人所有
2. 投标人须保证所提供软件不侵犯第三方知识产权

### 5.7 保密条款

投标人须对项目相关技术和业务信息严格保密，保密期限为合同终止后3年。

## 第六章 投标文件要求

### 6.1 投标文件组成

1. 商务部分：资格证明文件、商务响应文件
2. 技术部分：技术方案、项目团队、实施计划
3. 价格部分：报价一览表、分项报价表

1. 投标文件须为PDF格式，加盖公章
2. 电子投标文件须通过政府采购平台上传
3. 纸质投标文件须密封递交

### 6.3 签字盖章要求

1. ★投标文件须加盖投标人公章和法定代表人签字（或盖章）
2. 报价表须加盖公章和造价人员签字

### 6.4 废标条款

出现以下情形之一的，按无效投标处理：

1. 投标报价超过采购预算的
2. 未按规定交纳投标保证金的
3. 投标文件未按规定签字盖章的
4. 未响应实质性技术要求的（带★号条款为实质性要求）
5. 投标文件递交时间超过截止时间的
6. 投标人资格条件不符合要求的
