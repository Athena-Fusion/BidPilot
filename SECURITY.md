# Security Policy

## Supported Versions

BidPilot 当前为 Hackathon 演示项目，主分支 `main` 作为唯一维护版本。

## Reporting a Vulnerability

如果发现安全问题，请通过 GitHub Issue 说明复现步骤、影响范围和建议修复方式。请不要公开真实招标文件、企业资质、证书、人员或客户敏感信息。

## Data Safety

- 示例招标文件必须使用虚构数据。
- 上传文件默认只用于本地分析演示，不应包含真实敏感资料。
- 模型接口切换为真实模式时，使用者需要自行确认数据出境、保密和合规要求。
- 系统输出仅用于投标准备辅助，不构成法律、财务或最终投标意见。
